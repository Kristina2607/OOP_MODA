# Online Store - Moda

A C++ console application for e-commerce management focused on buying and selling clothes and fashion products.

## Purpose

This project was developed to demonstrate practical software design, role-based system behavior, business logic implementation, and custom data structure development in a larger C++ console application.

## Project Scope

The system is built around three main user roles:

- **Client**
  - view, sort, and purchase products
  - manage cart operations and discounts
  - complete checkout and track order history
  - rate purchased items and request refunds

- **Business Profile**
  - manage product listings
  - approve or reject pending orders
  - handle refund requests
  - review sales and revenue-related information

- **Administrator**
  - issue checks to clients
  - monitor financial transactions
  - access customer-related insights and statistics
    
## Highlights

- Built with **C++** and object-oriented design
- Supports multiple user roles: **Client**, **Business Profile**, and **Administrator**
- Includes core e-commerce functionality such as:
  - product browsing and sorting
  - shopping cart management
  - checkout and order processing
  - order history and product rating
  - refund requests and transaction tracking
- Implements **custom `Vector` and `String` classes**
- Developed **without STL**
- Supports **save/load functionality** for persistent application state
- Applies **OOP principles** and **design patterns** to organize the system
